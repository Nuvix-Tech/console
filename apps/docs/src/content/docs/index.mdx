# Welcome to Nuvix

Nuvix is a high-performance, open-source backend platform for modern applications.  
Built for speed, designed for scalability, and developer-first.

⚠️ **Warning:** Nuvix is under active development and **not production-ready**.

This documentation will guide you through:

- Getting started quickly
- Understanding our unique schema system
- Authentication and messaging
- Storage and file management
- API reference and examples
- Contributing to the project

Explore each section to get the most out of Nuvix.
