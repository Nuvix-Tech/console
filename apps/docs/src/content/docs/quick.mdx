## Get Started

<Steps>
## 1. Create Project

Sign up at [nuvix.in](https://www.nuvix.in) and create your first project.

## 2. Install SDK

```bash
npm install nuvix
```

## 3. Initialize Client

```javascript
import { Client } from 'nuvix';

const client = new Client()
    .setEndpoint('https://api.nuvix.in/v1')
    .setProject('<PROJECT_ID>');
```

## 4. Add Authentication

```javascript
import { Account } from 'nuvix';

const account = new Account(client);

// Create user
await account.create('user@example.com', 'password', 'User Name');

// Login
await account.createEmailPasswordSession('user@example.com', 'password');
```
</Steps>
