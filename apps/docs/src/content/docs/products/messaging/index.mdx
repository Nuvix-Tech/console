# Messaging

Nuvix Messaging helps you communicate with your users through push notifications, emails, and SMS text messages.

Sending personalized communication for marketing, updates, and realtime alerts can increase user engagement and retention. You can also use Nuvix Messaging to implement security checks and custom authentication flows.
