# MSG91

MSG91 lets you send customized SMS messages to your users. These SMS messages can be sent immediately or scheduled. You can send SMS messages for purposes like reminders, promotions, announcements, and even custom authentication flows.

## 1. Add provider

To add MSG91 as a provider, navigate to **Messaging** > **Providers** > **Add provider** > **SMS**.

<Frame>
<img src="/images/docs/messaging/providers/msg91/provider.png" alt="Add MSG91 provider configuration" />
</Frame>

Give your provider a name > choose **MSG91** > click **Save and continue**. The provider will be saved to your project, but not enabled until you complete its configuration.

## 2. Configure provider

In the **Configure** step, you will need to provide details from your MSG91 dashboard to connect your Nuvix project.

You will need to provide the following information from your **MSG91 dashboard**.

| Field name | Description |
|------------|-------------|
| **Auth key** | Click to open the Username dropdown > **Authkey** > **Verify your mobile number** > **Create Authkey**. |
| **Sender ID** | Head to MSG91 dashboard > **SMS** > **Sender ID** > **Create sender ID**. |
| **Sender number** | Your registered sender number with MSG91. |

After adding the following details, click **Save and continue** to enable the provider.

## 3. Test provider

Before sending your first message, make sure you've configured [a topic](/products/messaging/topics) and [a target](/products/messaging/targets) to send messages to.

<Tabs>
<Tab title="Console">
To send a test message, navigate to **Messaging** > **Messages** > **Create message** > **SMS**.

<Frame>
<img src="/images/docs/messaging/messages/create-sms-message.png" alt="Create SMS message interface" />
</Frame>

Add your message and in the targets step, select one of your test targets. Set the schedule to **Now** and click **Send**.

Verify that you can receive the message in your inbox. If not, check for logs in the Nuvix Console or in your provider's logs.
</Tab>

<Tab title="Server SDK">
To send a message programmatically, use the Nuvix Server SDK.

<CodeGroup>
```javascript
import { Client } from '@nuvix/client';

const nx = new Client()
  .setEndpoint('https://api.nuvix.in/v1')
  .setProject('<PROJECT_ID>')
  .setKey('<API_KEY>');

const message = await nx.messaging.createSms(
  '<MESSAGE_ID>',
  '<CONTENT>',
  [],
  [],
  [],
  true,
  ''
);
```
</CodeGroup>
</Tab>
</Tabs>

You can follow the [Send SMS messages](/products/messaging/send-sms-messages) journey to send your first SMS and test your provider.

## 4. Manage provider

<Tabs>
<Tab title="Console">
You can update or delete a provider in the Nuvix Console.

Navigate to **Messaging** > **Providers** > click your provider. In the settings, you can update a provider's configuration or delete the provider.
</Tab>

<Tab title="Server SDK">
To update or delete providers programmatically, use the Nuvix Server SDK.

<CodeGroup>
```javascript
import { Client } from '@nuvix/client';

const nx = new Client()
  .setEndpoint('https://api.nuvix.in/v1')
  .setProject('<PROJECT_ID>')
  .setKey('<API_KEY>');

// update provider
const provider = await nx.messaging.updateMsg91Provider(
  '<PROVIDER_ID>',
  '<NAME>',
  false,
  '<SENDER_ID>',
  '<AUTH_KEY>',
  '<FROM>'
);

// delete provider
await nx.messaging.deleteProvider('<PROVIDER_ID>');
```
</CodeGroup>
</Tab>
</Tabs>
