Update the currently logged in user's phone number. After updating the phone number, the phone verification status will be reset. A confirmation SMS is not sent automatically, however you can use the [POST /account/verification/phone](https://docs.nuvix.in/references/cloud/client-web/account#createPhoneVerification) endpoint to send a confirmation SMS.
